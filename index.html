What is toku?

An application for designing and running behavioral studies

Toku (better application name to come; github.com/nyucdsc/toku) provides a simple and fast solution for behavioral scientists to design, animate, and run a wide range of behavioral experiments. Built-in features allow researchers to design most studies without any prior programming experience, and for the few cases that built-in features do not meet study-specific needs, the tool is 100% open-source and customizable using standard web technologies (JavaScript, HTML/CSS). Studies designed with this tool can be run on devices that even young kids are familiar with (e.g., iPads, tablets, and even phones), and can be launched anywhere with an internet connection (an offline mode is available, too). That means families can complete studies from the comfort of their home, as if they were using any other educational app. Studies are shareable with other researchers, providing a smooth process for collaboration and study replication.

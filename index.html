What is psyStudio?

An application for designing and running behavioral studies

psyStudio (currently located at: github.com/nyucdsc/toku) provides a simple and fast solution for behavioral scientists to design, animate, and run a wide range of behavioral experiments. Built-in features allow researchers to design most studies without any prior programming experience, and for the few cases that built-in features do not meet study-specific needs, the tool is 100% open-source and customizable using standard web technologies (JavaScript, HTML/CSS). Studies designed with this tool can be run on devices that even young kids are familiar with (e.g., iPads, tablets, and even phones), and can be launched anywhere with an internet connection (an offline mode is available, too). That means families can complete studies from the comfort of their home, as if they were using any other educational app. Studies are shareable with other researchers, providing a smooth process for collaboration and study replication.



Getting started with psyStudio

Required dependencies

NodeJS
Before you can use psyStudio, you must have nodeJS installed on your computer. You can download nodeJs here. You should use either the Windows 64-bit installer or the macOS installer, depending on your computer.

Optional tools

Git/GitHub

While it is not required to use Git when working with psyStudio, we highly recommend using Git (and GitHub) for managing your studies with psyStudio. Follow these instructions on how to install Git onto your computer.

VS Code (or a text editor of your choice)

psyStudio does not require any prior knowledge on how to program, but the application (in its current state) requires users to configure the study within a markdown file. This is most easily done using a text editor. We recommend VS Code as a great option for doing this; it also has an graphic interface that makes it easier to use Git/GitHub! You can download VS Code here. instructions on how to install Git onto your computer.

Downloading psyStudio onto your computer for the first time

Download a copy of the psyStudio from GitHub.
Extract the zipped folder onto your computer.
Rename the extracted folder with the name of your study. E.g., "study-app"
